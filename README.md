# E-Summit BPDC

Official GitHub organization for **E-Summit, BITS Pilani Dubai Campus**.

This space hosts the codebases and technical assets used to build, manage, and deploy digital platforms for E-Summit BPDC.

---

## About

E-Summit is the flagship entrepreneurial event of BITS Pilani, Dubai Campus, aimed at promoting innovation, startups, and entrepreneurial thinking through talks, competitions, workshops, and networking.
